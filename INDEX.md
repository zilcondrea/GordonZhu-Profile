# Welcome to Your Portfolio Website

## 🎯 You're ready to deploy!

This folder contains a complete, production-ready portfolio website for recruiting purposes.

### ⚡ Quick Setup (1 hour)

1. **Get Firebase Credentials** (5 min)
   - Visit console.firebase.google.com
   - Create project "guoxuan-portfolio"
   - Copy config to `firebase-config.js`

2. **Configure** (5 min)
   - Update `firebase-config.js`
   - Change password in `admin/admin.js`
   - Get DeepSeek API key

3. **Test Locally** (5 min)
   ```bash
   firebase serve
   ```

4. **Deploy** (10 min)
   ```bash
   firebase deploy
   ```

5. **Add Content** (30 min)
   - Visit admin panel
   - Upload projects
   - Add resume
   - Share with recruiters
